Our Project

Our group decided to try to solve the problem with not being able to find a parking spot.

How?

There were many different ways we thought to solve this but the direction we decided upon is to put cameras with raspberry pi into the parking lots. Then have the camera to take pictures of the lot at many different angles, feed those pictures into an edge detection program to tell if there is a car in a spot. Then feed the data into a web app. The whole goal is to be able to open a web app select the lot then see the open parking spots.

Background information

A raspberry pi is a small credit card sized computer. It uses a SoC: Broadcom BCM2711B0 quad-core A72 (ARMv8-A) 64-bit @ 1.5GHz cpu, a GPU: Broadcom VideoCore VI, for Networking it has a 2.4 GHz and 5 GHz 802.11b/g/n/ac wireless LAN. For random access memory 1GB, 2GB, or 4GB LPDDR4 SDRAM. And Bluetooth 5.0.
